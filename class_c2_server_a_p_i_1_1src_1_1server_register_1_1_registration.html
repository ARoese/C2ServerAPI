Detailed Description

Represents a registration of this chivalry server with a server browser.

This class automatically manages heartbeats and information-sharing with a chivalry server list. updates to this object (such as map/player count changes) are automatically sent to the server list specified in the constructor. Heartbeat signals are also sent periodically at the interval(s) requested by the server.
